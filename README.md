# Fitness Tracker App

An all-in-one fitness application that tracks:
- Calories (nutrition)
- Workouts (sets, reps)
- Running (GPS tracking)

## Tech Stack
- Frontend: React Native (Expo)
- Backend: Node.js + Express
- Database: PostgreSQL
- Auth: JWT

## Planned Features
- User authentication
- Dashboard
- Workout tracking
- Nutrition tracking
- Running tracker
- AI recommendations (future)
